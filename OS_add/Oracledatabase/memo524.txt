在Linux 8.8上，oracle-database-preinstall是一个预安装包，用于在Oracle Linux系统上自动配置和安装Oracle数据库安装所需的操作系统包和系统设置。这个预安装包会安装所有Oracle Grid Infrastructure和Oracle数据库安装所需的额外RPM软件包，并解决任何依赖关系。

Oracle Database Preinstallation RPM可以自动执行许多手动预安装任务，如添加用户和组、设置内核参数、安装必需的软件包等。这有助于简化Oracle数据库的安装过程。

要安装oracle-database-preinstall包，你可以使用以下命令：

bash
sudo dnf install oracle-database-preinstall-21c
如果你的系统上没有与你的Oracle数据库和Oracle Grid Infrastructure版本当前的Oracle数据库预安装RPM软件包选项，那么在完成操作系统安装后，你将需要手动安装你版本的RPM。

安装完成后，Oracle Linux会自动创建一个标准的（非角色分配的）Oracle安装所有者和组，并设置其他内核配置设置，这些设置是Oracle安装所需的。
